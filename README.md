# ARAM Archipelago Tracker

A Vue 3 + TypeScript live session app for ARAM players. Join a local WebSocket session, track checks as friends play, and unlock new champions in real time.

## Run locally

1. Install dependencies:
   - `npm install`
2. Start the live session server:
   - `npm run server`
3. Start the app:
   - `npm run dev`

Alternatively, run both together:

- `npm run dev:full`
